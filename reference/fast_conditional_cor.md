# Fast Conditional Correlation

Compute correlation conditioned on a third variable being in a range.

## Usage

``` r
fast_conditional_cor(x, y, z, z_min, z_max)
```

## Arguments

- x:

  Numeric vector

- y:

  Numeric vector

- z:

  Conditioning variable

- z_min:

  Lower bound for z

- z_max:

  Upper bound for z

## Value

Correlation coefficient
